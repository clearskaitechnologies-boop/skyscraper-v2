# Demo Video Directory

## Upload Instructions

Place your demo video here as `intro.mp4` or `demo.mp4`.

### Recommended Video Specifications

- **Format:** MP4 (H.264 codec)
- **Resolution:** 1920x1080 (1080p) or 1280x720 (720p)
- **Duration:** 60-90 seconds
- **File Size:** < 50MB (optimize for web)
- **Frame Rate:** 30fps or 60fps

### Video Content Suggestions

1. **Opening (5s):** SkaiScraper logo + tagline
2. **Dashboard Overview (15s):** Show pro dashboard with claims/leads
3. **AI Tools Demo (30s):** Generate estimate, rebuttal, or weather report
4. **Client Portal (15s):** Show client view and communication
5. **Closing (10s):** CTA - "Get Started Free" or "Schedule Demo"

### Alternative: Use YouTube/Vimeo

If hosting externally:

1. Upload to YouTube or Vimeo
2. Copy the video ID
3. Update homepage with `<VideoPlayer youtubeId="YOUR_ID" />`
