## Logo Don'ts (Misuse Examples)

### ‚ùå **DO NOT:**

1. **Stretch or distort logo**
   - Always maintain aspect ratio
   - Scale proportionally only

2. **Change colors outside brand palette**
   - No random colors (red, green, purple, etc.)
   - Only use White, Electric Cyan, Midnight Navy

3. **Add effects without approval**
   - No gradients, drop shadows (except approved glassmorphic styles)
   - No 3D effects, bevels, embossing

4. **Place on busy backgrounds without treatment**
   - Always ensure high contrast
   - Use glassmorphic panel if background is complex

5. **Rotate or angle the logo**
   - Logo should always be horizontal
   - Exception: Vertical stacking for narrow spaces (with approval)

6. **Crowd the logo**
   - Respect minimum clear space at all times

7. **Recreate or modify the logo**
   - Always use official logo files
   - Do not attempt to retype or redraw

8. **Use low-resolution files**
   - Always use vector (.svg, .eps) when possible
   - Use high-res PNG (300 DPI) for print

## Logo File Formats Explained

| Format   | Usage                           | Best For                                      |
| -------- | ------------------------------- | --------------------------------------------- |
| **.SVG** | Vector (scalable)               | Web, presentations, any digital use           |
| **.EPS** | Vector (scalable)               | Print, Adobe Illustrator, professional design |
| **.PNG** | Raster (transparent background) | Web, social media, PowerPoint                 |
| **.JPG** | Raster (solid background)       | Email, basic web (avoid when possible)        |
| **.ICO** | Icon format                     | Favicons, Windows icons                       |
| **.PDF** | Vector (print-ready)            | Print materials, large format                 |

**Recommended Workflow:**

- **Design/Edit:** Use .SVG or .EPS (vector)
- **Web:** Use .SVG (preferred) or .PNG (fallback)
- **Print:** Use .EPS or high-res .PNG (300 DPI)
- **Presentations:** Use .SVG or .PNG
- **Social Media:** Use .PNG with transparent background

## Partner Co-Branding Rules

When displaying SkaiScraper logo alongside partner logos:

1. **Equal Prominence:**
   - All logos should be similar size (¬±20% variance)
   - Maintain equal clear space around each logo

2. **Alignment:**
   - Center-align vertically
   - Space evenly horizontally

3. **Separator (Optional):**
   - Use subtle vertical line OR "+" symbol between logos
   - Separator color: White or Cyan (low opacity)

4. **Background:**
   - Use neutral background that works for all logos
   - Prefer dark backgrounds (consistent with SkaiScraper brand)

**Example Lockup:**

```
[Partner Logo]  +  [SkaiScraper Logo]
```
