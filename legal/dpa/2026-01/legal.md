# Data Processing Agreement (DPA)

**ClearSkai Technologies, LLC (DBA "SkaiScraper")**  
Prescott, Arizona  
Version: 2026-01  
Effective Date: January 2026

## 1. Roles

Customer = Data Controller  
SkaiScraper = Data Processor

## 2. Processor Obligations

Processor shall:

- Process data only according to Controller's instructions
- Implement appropriate security measures
- Notify of data breaches within 72 hours

## 3. Use of Data for AI Processing

Processor uses personal data for generating automated summaries and reports. Processor **DOES NOT** use Controller's data to train general-purpose AI models.
