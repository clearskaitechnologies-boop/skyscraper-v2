# SkaiScraper™ Mockup Prompts

This folder contains exact prompts and metadata for generating the six photoreal mockups (A–F) at 1920×1200, sRGB, matte print style.

Usage:

- Use your preferred image model (Stable Diffusion XL, Midjourney, DALL·E 3, or a commercial API).
- Render size: 1920×1200, sRGB, matte print finish, photoreal.
- Filenames are given in each prompt file. Do not change the brand or tagline text.

Store outputs in `public/mockups/` and commit or upload to your CDN/storage bucket for the app to use.
